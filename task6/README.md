# Task 6: House Price Prediction

## Objective
Predict house prices based on property features such as size, bedrooms, and location.

## Dataset
- Source: House Price Prediction Dataset (Kaggle)
- Features: square footage, number of bedrooms, bathrooms, location, etc.
- Target: price

## Models Applied
- Linear Regression

## Key Results and Findings
- Model evaluated using MAE and RMSE
- Feature scaling with StandardScaler improved model performance
- Most influential features identified via model coefficients
- Scatter plot of actual vs predicted prices shows strong linear correlation
