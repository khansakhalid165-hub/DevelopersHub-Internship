# Task 1: Iris Dataset Visualization

## Objective
Explore and visualize the Iris dataset to understand feature distributions and relationships between species.

## Dataset
- Iris Dataset (built-in from sklearn)
- 150 samples, 3 classes (Setosa, Versicolor, Virginica)
- Features: sepal length, sepal width, petal length, petal width

## Models Applied
- No model — this is an EDA and visualization task

## Key Results and Findings
- Petal length and petal width are the most distinguishing features between species
- Setosa is clearly separable from the other two classes
- Versicolor and Virginica show some overlap in features
