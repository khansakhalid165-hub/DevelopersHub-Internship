# Task 2: Stock Price Prediction

## Objective
Predict the next day's closing price of Apple (AAPL) stock using historical price data.

## Dataset
- Source: Yahoo Finance via yfinance library
- Stock: Apple Inc. (AAPL)
- Period: 5 years, daily interval
- Features: Open, High, Low, Volume

## Models Applied
- Linear Regression

## Key Results and Findings
- Model successfully tracks the overall price trend
- Predicted prices closely follow actual closing prices
- Same-day High and Low are the strongest predictors of closing price
